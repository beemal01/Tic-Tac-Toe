# Tic-Tac-Toe
A classic **Tic-Tac-Toe** game built using **HTML, CSS, and JavaScript**. Players take turns marking **X** or **O** on a 3Ã—3 grid, aiming to get three in a row **horizontally, vertically, or diagonally**. The game features **interactive UI, real-time win detection, and a reset option**. **JavaScript** handles player moves, game logic, and result announcements, while **CSS** enhances the visual design.
